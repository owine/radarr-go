This is a Go-based implementation of Radarr with API compatibility.

Available Endpoints:

  • GET /ping - Health check
  • GET /api/v3/system/status - System status
  • GET /api/v3/movie - List all movies
  • GET /api/v3/movie/:id - Get specific movie
  • POST /api/v3/movie - Add new movie
  • PUT /api/v3/movie/:id - Update movie
  • DELETE /api/v3/movie/:id - Delete movie
  • GET /api/v3/moviefile - List movie files
  • GET /api/v3/search/movie?term=query - Search movies

Quick Test:

  • curl http://localhost:7878/api/v3/system/status
  • curl http://localhost:7878/api/v3/movie
